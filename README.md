# GPU Market Tracker

Real-time GPU price tracker for Chilean and international markets. Built with NestJS and Next.js.

## Tech Stack:
- **Backend:** NestJS (Node.js) + PostgreSQL
- **ORM:** Prisma
- **Frontend:** Next.js (React) + Tailwind CSS

## Key Features:
- Automated tracking for Chilean retailers and International market (Amazon).
- Time-series visualization and historical price tracking.
- AI Assistan for smart purchase recommendations based on real-time data.
